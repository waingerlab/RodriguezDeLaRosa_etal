# RodriguezDeLaRosa_etal

# These scripts take TIF files from the IMX microconfocal, assemble them into a video, and analyzes GCaMP activity over time. To use these scripts, open "Muscle_GCaMP_250716AH.m" and make sure the "addpath" function points to the correct directory of the two subfolders ("Graphs" and "Universal_functions"). Then ensure that the glob function points to the location of the TIF files and run the remainder of the script.

# Cells were prepared by Alejandra Rodriguez de la Rosa, and cells were imaged by both ALejandra Rodriguez de la Rosa and Aaron Held. Aaron Held wrote the image analysis scripts to analyze the GCaMP data.
